# 安培力方向关系学生探究工具

面向上海市延安中学课堂使用的学生端静态网页工具。学生输入电流方向、电磁场方向角 `α`、安培力方向角 `β` 后，可以观察 `I`、`B`、`F` 在空间中的方向关系，切换 Z-Y 正视图和可旋转三维视角，并尝试归纳左手定则。

## 项目用途

- 课堂学生端网页，适合通过链接或二维码访问。
- 只保留安培力方向关系的三维探究功能。
- 不包含教师端的 DIS、OCR、摄像头、屏幕捕获、CSV 导出和关键图像记录功能。

## 本地运行

```bash
npm install
npm run dev
```

构建发布包：

```bash
npm run build
```

构建产物目录为：

```text
dist
```

## 数据输入

右侧输入区包含：

- 电流方向：`I: +x` 或 `I: -x`
- `∠α`：B 磁场方向角，范围 `0°` 到 `360°`
- `∠β`：F 安培力方向角，范围 `0°` 到 `360°`

点击“点击绘图”后，左侧三维区域会根据当前输入重新绘制导体棒、电流方向、磁场方向、安培力方向和角度弧线。

## α、β 的定义

`α` 和 `β` 都在 y-z 平面内定义，参考方向固定为 `+y`：

- `0°` 表示方向沿 `+y`
- `90°` 表示方向沿 `+z`
- `180°` 表示方向沿 `-y`
- `270°` 表示方向沿 `-z`

方向向量换算：

```ts
const alphaRad = alphaDeg * Math.PI / 180;
const bDir = new Vector3(0, Math.cos(alphaRad), Math.sin(alphaRad));

const betaRad = betaDeg * Math.PI / 180;
const fDir = new Vector3(0, Math.cos(betaRad), Math.sin(betaRad));
```

## 视角和探究按钮

页面下方提供：

- Z-Y 正视图
- 可旋转视角
- 显示 / 隐藏 αβ
- 显示 / 隐藏 B-F 夹角验证
- 显示 / 隐藏左手定则提示
- 全屏展示

三维区域支持鼠标和触摸旋转、缩放。默认进入 Z-Y 正视图，便于理解角度定义。

## Netlify 部署

Netlify 配置建议：

- Build command: `npm run build`
- Publish directory: `dist`

本项目是静态网页，不需要服务器、数据库、登录、摄像头权限或屏幕捕获权限。

## 与教师端项目的关系

本学生端项目参考了教师端右侧三维图示的坐标约定和视觉表达，但作为独立项目实现。它不依赖教师端的 DIS、OCR、Tesseract、摄像头、ROI、实时数据总览、调试工具或 CSV 导出代码。
